# SS14 ECS Basics

Use this skill when a task is more about understanding the SS14 ECS mental model than about one specific subsystem.

## Workflow

1. Open `references/ecs-primer.md`.
2. Open `references/simple-system-walkthrough.md` for a concrete code-reading path.
3. Open `references/event-reading-guide.md` when explaining subscriptions and handlers.
4. Keep the component-data/system-behavior split strict.
