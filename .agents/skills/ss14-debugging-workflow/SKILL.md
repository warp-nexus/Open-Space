# SS14 Debugging Workflow

Use this skill when the task is investigation-first.

## Workflow

1. Open `references/debugging-mindset.md`.
2. Open `references/logs-vv-breakpoints.md` for the main tools.
3. Open `references/console-and-runtime-tools.md` for runtime commands and inspection helpers.
4. Reduce the bug to one failing step before “fixing” anything.
