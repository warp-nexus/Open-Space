# Visual Prototype Anchors

## Foldable

- `Resources/Prototypes/Entities/foldable.yml`
- Shows a compact `GenericVisualizer` with folded and unfolded layers.

## Bag Open Or Close

- `Resources/Prototypes/Entities/Objects/base_item.yml`
- Shows `Appearance` plus `GenericVisualizer` for open and closed bag layers.

## Paper Label Visuals

- `Resources/Prototypes/Entities/Objects/base_item.yml`
- Shows how a reusable visualized parent can define label visibility.

## Reminder

When the prototype already demonstrates the visual state you need, mirror that pattern instead of inventing a parallel layer scheme.
