# SS14 Prototype Basics

Use this skill for first-contact prototype work and for explaining prototypes to contributors or agents.

## Workflow

1. Open `references/prototype-primer.md`.
2. Open `references/first-prototype-workflow.md` for the practical search-parent-edit loop.
3. Open `references/common-prototype-components.md` when you need quick reminders for the most common YAML-side components.
4. Start from an existing nearby prototype instead of writing a large entity from scratch.
