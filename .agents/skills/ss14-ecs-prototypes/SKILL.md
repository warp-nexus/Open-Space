# SS14 ECS Prototypes

Use this skill for prototype layout, YAML structure, file placement, and code-to-data pairing.

## Workflow

1. Open `references/prototype-structure.md`.
2. Open `references/prototype-examples.md` for the quick map.
3. Open the narrower example file you actually need:
   - `references/item-and-structure-examples.md`
   - `references/reagent-examples.md`
   - `references/generic-visualizer-examples.md`
   - `references/localized-prototype-examples.md`
3. Put prototypes in the most specific existing subtree.
4. Add or update localization and assets alongside the prototype change.
5. Run YAML validation after edits.
