Use this skill when a task imports external work or needs repository-specific license guidance.

## Workflow

1. Open `references/porting-workflow.md`.
2. Open `references/repository-license-model.md`.
3. Open `references/attribution-and-safety.md`.
4. Do not import code or assets until the license and attribution path are understood.
