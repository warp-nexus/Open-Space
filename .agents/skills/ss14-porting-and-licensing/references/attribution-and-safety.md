# Attribution And Safety

## Required Checks

- known source
- compatible license
- correct attribution path
- no hidden AGPL or other incompatible carryover unless explicitly allowed by local policy

## Contribution Rule

- `CONTRIBUTING.md` says that submitting a PR means agreeing to the local `CLA.TXT`

## Good Practice

- keep a note of the source PR, repo, or commit
- mention the source and what was adapted in the final PR description
- do not mix a large port with unrelated cleanup
